.. _wulfric_cite:

***************
How to cite us?
***************

If you use wulfric in a published work, please cite in the following form:

.. code-block::

    wulfric v<version>, wulfric.org.

BibTeX entry:

.. code-block:: LaTeX

    @online{wulfric,
        author = {Andrey Rybakov},
        title  = {wulfric v<version>},
        year   = {2023},
        url    = {wulfric.org},
    }
