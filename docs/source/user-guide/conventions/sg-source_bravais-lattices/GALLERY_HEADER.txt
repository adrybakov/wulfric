****************
Bravais lattices
****************

Wulfric implements two conventions for the definition of the high-symmetry points in
reciprocal space and recommended k-paths

*   "HPKOT" as defined in [1]_

    In this work authors defined a set of high-symmetry points in
    reciprocal space and recommended k-paths for each "extended Bravais lattice symbol".
    Therefore, we provide in total 29 examples for each extended Bravais lattice symbol.

*   "SC" as in [2]_

    In this work authors defined a set of high-symmetry points in
    reciprocal space and recommended k-paths for each variation of Bravais lattice.
    We provide examples for each of 25 lattice variations.
