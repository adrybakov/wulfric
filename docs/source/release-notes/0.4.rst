.. warning::

Whats new?

Dependency on the |spglib|_ was introduced.

0.4.0
-----
The changes are described above and:

* Add :py:class:`.StandardizationTypeMismatch` exception.
* Function ``Kpoints.coordinates()`` is removed, use :py:meth:`.Kpoints.ticks()` instead.
